# Guides index

Runbooks and notes live as **`prompts/guides/<name>.md`** (stable paths for matrix/designer links).

**Canonical metadata:** [`INVENTORY.md`](INVENTORY.md) — every guide’s **suggested status** (active / pending / finished) and path. Update that file when priorities change; **no need** to move files on disk unless you want to reorganize paths deliberately.

If you still have empty `active/`, `pending/`, or `finished/` folders from an earlier layout, you can delete those folders — they are not part of the current model.
